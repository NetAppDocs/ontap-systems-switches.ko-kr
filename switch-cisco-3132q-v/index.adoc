= Cisco 3132Q-V 스위치
:icons: font
:imagesdir: ../media/


[role="lead"]
Cisco 3132Q-V 스위치를 AFF 또는 FAS 클러스터의 클러스터 스위치로 사용할 수 있습니다. 이 절차를 사용하여 스위치를 설치하고, 기존 스위치에서 마이그레이션하고, 스위치를 교체하고, RCF 파일을 스위치에서 업데이트할 수 있습니다.
