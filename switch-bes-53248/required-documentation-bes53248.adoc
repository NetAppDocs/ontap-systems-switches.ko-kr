= BES-53248 클러스터 스위치에 대한 문서 요구 사항
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
BES-53248 스위치 설치 시에는 해당 스위치 및 컨트롤러 설명서를 반드시 참조하십시오.



== Broadcom 문서

BES-53248 클러스터 스위치를 설정하려면 Broadcom 지원 사이트에서 제공되는 다음 문서가 필요합니다link:https://www.broadcom.com/support/bes-switch["Broadcom 이더넷 스위치 제품군"^].

[cols="1,2"]
|===
| 문서 제목 | 설명 


 a| 
_EFOS 관리자 가이드 v3.4.3_
 a| 
일반적인 네트워크에서 BES-53248 스위치를 사용하는 방법에 대한 예를 제공합니다.



 a| 
_EFOS CLI 명령 참조 v3.4.3_
 a| 
BES-53248 소프트웨어를 보고 구성하는 데 사용하는 명령줄 인터페이스(CLI) 명령을 설명합니다.



 a| 
_EFOS 시작 가이드 v3.4.3_
 a| 
BES-53248 스위치에 대한 자세한 정보를 제공합니다.



 a| 
_EFOS SNMP 참조 가이드 v3.4.3_
 a| 
일반적인 네트워크에서 BES-53248 스위치를 사용하는 방법에 대한 예를 제공합니다.



 a| 
_EFOS 스케일링 매개변수 및 값 v3.4.3_
 a| 
지원되는 플랫폼에서 EFOS 소프트웨어가 제공되고 검증되는 기본 스케일링 매개변수를 설명합니다.



 a| 
_EFOS 기능 사양 v3.4.3_
 a| 
지원되는 플랫폼의 EFOS 소프트웨어 사양을 설명합니다.



 a| 
_EFOS 릴리스 노트 v3.4.3_
 a| 
BES-53248 소프트웨어에 대한 릴리스별 정보를 제공합니다.



 a| 
_클러스터 네트워크 및 관리 네트워크 호환성 매트릭스_
 a| 
네트워크 호환성에 대한 정보를 제공합니다. 매트릭스는 link:https://mysupport.netapp.com/site/products/all/details/broadcom-cluster-switches/downloads-tab["Broadcom 클러스터 스위치"^]의 BES-53248 스위치 다운로드 사이트에서 확인할 수 있습니다.

|===
