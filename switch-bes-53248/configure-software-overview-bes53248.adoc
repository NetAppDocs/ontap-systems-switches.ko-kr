= BES-53248 스위치용 소프트웨어 설치 워크플로
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
BES-53248 클러스터 스위치용 소프트웨어를 설치하고 구성하려면 다음 단계를 따르세요.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["하나"]link:configure-install-initial.html["스위치 구성"]
[role="quick-margin-para"]
BES-53248 클러스터 스위치를 구성합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["둘"]link:configure-efos-software.html["EFOS 소프트웨어를 설치하세요"]
[role="quick-margin-para"]
BES-53248 클러스터 스위치에 Ethernet Fabric OS(EFOS) 소프트웨어를 다운로드하여 설치합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["삼"]link:configure-licenses.html["BES-53248 클러스터 스위치에 대한 라이센스 설치"]
[role="quick-margin-para"]
선택적으로, 추가 라이선스를 구매하고 설치하여 새로운 포트를 추가할 수 있습니다.  스위치 기본 모델은 16개의 10GbE 또는 25GbE 포트와 2개의 100GbE 포트에 대한 라이선스를 받았습니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["네"]link:configure-install-rcf.html["참조 구성 파일(RCF) 설치"]
[role="quick-margin-para"]
BES-53248 클러스터 스위치에 RCF를 설치하거나 업그레이드한 다음, RCF가 적용된 후 추가 라이선스에 대한 포트를 확인합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["다섯"]link:configure-ssh.html["BES-53248 클러스터 스위치에서 SSH 활성화"]
[role="quick-margin-para"]
CSHM(Ethernet Switch Health Monitor) 및 로그 수집 기능을 사용하는 경우 스위치에서 SSH를 활성화하세요.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["육"]link:reset-switch-bes-53248.html["스위치를 공장 기본값으로 재설정합니다"]
[role="quick-margin-para"]
BES-53248 클러스터 스위치 설정을 지웁니다.
