= BES-53248 클러스터 스위치에 대한 업그레이드 워크플로
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
해당되는 경우 Broadcom BES-54328 클러스터 스위치에서 EFOS 소프트웨어와 참조 구성 파일(RCF)을 업그레이드하려면 다음 단계를 따르세요.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["하나"]link:upgrade-efos-software.html["EFOS 버전을 업그레이드하세요"]
[role="quick-margin-para"]
BES-53248 클러스터 스위치에 Ethernet Fabric OS(EFOS) 소프트웨어를 다운로드하여 설치합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["둘"]link:upgrade-rcf.html["RCF 버전을 업그레이드하세요"]
[role="quick-margin-para"]
BES-53248 클러스터 스위치에서 RCF를 업그레이드한 다음, RCF가 적용된 후 추가 라이선스에 대한 포트를 확인합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["삼"]link:replace-verify.html["업그레이드 후 ONTAP 클러스터 네트워크 확인"]
[role="quick-margin-para"]
BES-53248 클러스터 스위치의 EFOS 소프트웨어 또는 RCF를 업그레이드한 후 ONTAP 클러스터 네트워크의 상태를 확인합니다.
