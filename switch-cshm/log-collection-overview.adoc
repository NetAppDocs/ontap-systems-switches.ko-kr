= 로그 수집 개요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
로그 수집을 설정하면 AutoSupport 에서 수집하는 주기적 데이터의 매시간 수집을 활성화하고 자세한 지원 로그를 요청할 수 있습니다.

로그 수집 기능을 사용하면 스위치 로그 및 진단 데이터를 자동으로 수집하여 네트워크 스위치 문제를 해결할 수 있습니다. AutoSupport를 통해 예약된 일정에 따라 주기적으로 데이터를 수집하거나, 특정 문제 해결을 위한 상세 로그가 필요할 때 온디맨드 수집을 트리거할 수 있습니다. 수집된 로그는 스위치 상태, 구성 변경 사항 및 네트워크 이벤트를 파악하는 데 도움이 됩니다.

자세한 내용은 link:config-log-collection.html["로그 수집 구성"]을(를) 참조하십시오.
