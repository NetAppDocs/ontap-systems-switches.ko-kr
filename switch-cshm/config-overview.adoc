= 구성 개요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이더넷 스위치 상태 모니터링(CSHM)은 네트워크 스위치의 작동 상태를 확인하고 디버깅을 위해 스위치 로그를 수집하는 역할을 합니다.

CSHM을 사용하여 네트워크 스위치를 모니터링하려면 먼저 로그 수집을 구성해야 하며, 보안 강화를 위해 선택적으로 SNMPv3도 구성해야 합니다. CSHM이 스위치 로그 및 진단 데이터를 수집하려면 로그 수집이 필수적입니다. SNMPv3는 ONTAP과 스위치 간의 안전한 통신을 제공합니다.
