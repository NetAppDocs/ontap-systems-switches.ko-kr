= 로그 수집 문제 해결
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
로그 수집 기능에서 보고된 다음 오류 상태 중 하나가 발생하는 경우(출력에서 확인 가능) `system switch ethernet log show` 명령), 해당 디버그 단계를 시도하세요.

|===


| *로그 수집 오류 상태* | *해결* 


 a| 
*RSA 키가 없습니다*
 a| 
ONTAP SSH 키를 다시 생성합니다.



 a| 
*비밀번호 변경 오류*
 a| 
자격 증명을 확인하고, SSH 연결을 테스트하고, ONTAP SSH 키를 다시 생성합니다.  스위치 설명서를 검토하거나 NetApp 지원팀에 문의하여 지침을 받으세요.



 a| 
*FIPS에는 ECDSA 키가 없습니다*
 a| 
FIPS 모드가 활성화된 경우 다시 시도하기 전에 스위치에서 ECDSA 키를 생성해야 합니다.



 a| 
*기존 로그 발견*
 a| 
스위치에서 이전 로그 수집 파일을 제거합니다.



 a| 
*스위치 덤프 로그 오류*
 a| 
스위치 사용자에게 로그 수집 권한이 있는지 확인하세요.  위의 전제 조건을 참조하세요.

|===

NOTE: 해결 방법이 작동하지 않으면 NetApp 지원팀에 문의하세요.
