= 이더넷 스위치 상태 모니터링
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이더넷 스위치 상태 모니터링(CSHM)은 네트워크 스위치의 작동 상태를 확인하고 디버깅을 위해 스위치 로그를 수집하는 역할을 합니다.

CSHM은 클러스터 및 스토리지 네트워크 스위치를 모니터링하여 네트워크 인프라 안정성을 유지합니다. 상태 모니터링은 스위치 문제를 자동으로 감지하고, 문제 해결을 위한 로그를 수집하며, 네트워크 문제가 발생할 경우 경고를 제공합니다. CSHM은 다양한 스위치 제조사와 모델을 지원하므로 스토리지 환경에 영향을 미치기 전에 네트워크 연결 문제를 식별하고 해결할 수 있습니다.
