== 스위치 연결형 케이블 NS224 스토리지

NS224 스토리지를 스위치 연결로 케이블로 연결하려면 다음 스위치 연결 다이어그램을 따르세요.

image:9336c_image1.jpg["스위치 부착"]

를 참조하십시오 https://hwu.netapp.com/Switch/Index["Hardware Universe"] 스위치 포트에 대한 자세한 내용은 다음을 참조하세요.



== NS224 케이블 스토리지를 직접 연결하여 사용 가능

공유 스위치 스토리지 포트를 사용하는 대신 NS224 스토리지를 직접 연결하여 케이블로 연결하려면 다음 직접 연결 다이어그램을 따르세요.

image:9336c_image2.jpg["직접 부착"]

를 참조하십시오 https://hwu.netapp.com/Switch/Index["Hardware Universe"] 스위치 포트에 대한 자세한 내용은 다음을 참조하세요.



== Cisco Nexus 9336C-FX2 케이블링 워크시트

지원되는 플랫폼을 문서화하려면 완성된 샘플 케이블링 워크시트를 가이드로 사용하여 빈 케이블링 워크시트를 완성해야 합니다.

각 스위치 쌍의 샘플 포트 정의는 다음과 같습니다.

image:cabling_worksheet.jpg["케이블링 워크시트"]

어디:

* 100G ISL을 스위치 A 포트 35로
* 100G ISL을 스위치 A 포트 36으로
* 100G ISL을 스위치 B 포트 35로
* 100G ISL을 스위치 B 포트 36으로




== 빈 케이블링 워크시트

빈 케이블링 워크시트를 사용하여 클러스터에서 노드로 지원되는 플랫폼을 문서화할 수 있습니다.  Hardware Universe 의 지원되는 클러스터 연결 표는 플랫폼에서 사용되는 클러스터 포트를 정의합니다.

image:blank_cabling_worksheet.jpg["빈 케이블링 워크시트"]

어디:

* 100G ISL을 스위치 A 포트 35로
* 100G ISL을 스위치 A 포트 36으로
* 100G ISL을 스위치 B 포트 35로
* 100G ISL을 스위치 B 포트 36으로
