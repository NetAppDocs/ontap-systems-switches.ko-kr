[[hard-code-the-speed-on-breakout-ports]]
== 브레이크아웃 포트의 속도를 하드코딩합니다

스위치의 브레이크아웃 포트(4x100GbE)는 100GbE 속도로 하드코딩해야 합니다. 속도를 하드코딩하지 않으면 포트가 활성화되지 않고 오류 비활성화 상태가 됩니다.

예를 들어, 브레이크아웃 포트 1~13에 100GbE를 하드코딩하려면 구성 모드에서 다음 명령을 실행하십시오. 예시에서는 첫 번째 스위치(cs1)에 명령을 적용하는 방법을 보여줍니다. 두 번째 스위치에도 동일한 명령을 반복해서 실행하십시오.

[listing, subs="+quotes"]
----
(cs1)(config)# *interface e1/1/1-4, e1/2/1-4, e1/3/1-4, e1/4/1-4, e1/5/1-4, e1/6/1-4, e1/7/1-4, e1/8/1-4, e1/9/1-4, e1/10/1-4, e1/11/1-4, e1/12/1-4, e1/13/1-4*
(cs1)(config-if)# *speed 100000*
(cs1)(config-if)# *no negotiate auto*
(cs1)(config-if)# *exit*
(cs1)(config)# *exit*
(cs1)# *copy running-config startup-config*
----
