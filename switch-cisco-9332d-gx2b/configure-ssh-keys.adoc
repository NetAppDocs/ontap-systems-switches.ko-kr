= SSH 구성을 확인합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
CSHM(Ethernet Switch Health Monitor) 및 로그 수집 기능을 사용하는 경우 스위치에서 SSH 및 SSH 키가 활성화되어 있는지 확인하세요.

.단계
. SSH가 활성화되었는지 확인합니다.
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. SSH 키가 활성화되어 있는지 확인합니다.
+
`show ssh key`
