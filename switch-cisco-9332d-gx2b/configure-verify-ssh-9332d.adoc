= 로그 수집을 위한 SSH 확인 - Cisco Nexus 9332D-GX2B
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
새로운 AFX 1K 또는 AFX 2K 시스템 설치, 스위치 교체 또는 공장 기본 설정으로 재설정하는 과정에서 스위치를 구성하여 이더넷 스위치 상태 모니터링(CSHM) 로그 수집 기능을 사용하려면 스위치에서 SSH가 활성화되어 있는지 확인해야 합니다.

.시작하기 전에
직렬 콘솔을 사용하여 스위치에 연결해야 합니다.

.단계
. SSH가 활성화되어 있는지 확인하세요.
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. SSH 키가 활성화되어 있는지 확인하세요.
+
`show ssh key`

+

.다음은 무엇인가요?
* SSH를 통해 로그 수집이 활성화되어 있는지 확인한 후에는 link:../switch-cshm/cshm-overview.html["이더넷 스위치 상태 모니터링 구성"].
+
** 이더넷 스위치 상태 모니터링을 구성한 후 Cisco의 Smart Call Home 지원 기능을 활용하려면 link:configure-smart-call-9332d.html["스마트 콜 홈 설정"].


* 새로운 AFX 1K 또는 AFX 2K 시스템 설치의 일부로 스위치를 구성한 경우, link:install-complete-afx-setup-9332d.html["AFX 시스템 설정을 완료합니다"]해야 합니다.
