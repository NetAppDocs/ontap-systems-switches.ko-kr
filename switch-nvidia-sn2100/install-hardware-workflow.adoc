SN2100 클러스터 스위치의 하드웨어를 설치하고 구성하려면 다음 단계를 따르세요.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["하나"]link:install-hardware-sn2100-cluster.html["하드웨어 설치"]
[role="quick-margin-para"]
스위치 하드웨어를 설치합니다.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["삼"]link:install-cable-shelves-sn2100-cluster.html["NS224 선반에 케이블 연결"]
