= SSH 구성을 확인하세요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
CSHM(Ethernet Switch Health Monitor) 및 로그 수집 기능을 사용하는 경우 클러스터 스위치에서 SSH 및 SSH 키가 활성화되어 있는지 확인하세요.

.단계
. SSH가 활성화되어 있는지 확인하세요.
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. SSH 키가 활성화되어 있는지 확인하세요.
+
`show ssh key`

+
